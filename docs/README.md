## Overview

eSIM Manager is a production-ready Windows Desktop Application built with .NET 8 WPF that provides comprehensive enterprise eSIM lifecycle management capabilities. The application supports both WLAN and Bluetooth connectivity for managing eSIM profiles on devices.

## Key Features

- **Device Discovery**: Automatic discovery of devices via WLAN and Bluetooth
- **Connection Management**: Seamless device pairing and connection handling
- **eSIM Profile Management**: List, provision, activate, deactivate, and remove eSIM profiles
- **System Logging**: Comprehensive event logging and diagnostics
- **Offline Recovery Mode**: Continue operations when network connectivity is limited
- **Enterprise Configuration**: Secure enterprise configuration management
- **Modern UI**: Professional dark theme interface with intuitive navigation
