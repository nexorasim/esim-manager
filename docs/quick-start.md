# Quick Start Guide

Get started with eSIM Manager in minutes.

## First Launch

1. Launch eSIM Manager from your desktop shortcut or Start Menu
2. The application opens to the Dashboard view
3. You'll see the welcome screen with quick actions

## Connecting Your First Device

### Step 1: Navigate to Device Connection

Click **Device Connection** in the left sidebar.

### Step 2: Select Connection Type

Choose your preferred connection method:
- **WLAN**: For network-based device connectivity
- **Bluetooth**: For direct Bluetooth pairing

### Step 3: Discover Devices

1. Click the **Discover Devices** button
2. Wait for the scan to complete (5-30 seconds)
3. Available devices will appear in the list

### Step 4: Connect

1. Select a device from the list
2. Click **Connect Device**
3. Wait for the connection to establish
4. Status will update to "Connected to [Device Name]"

## Managing eSIM Profiles

### Viewing Profiles

1. Navigate to **eSIM Profiles** in the sidebar
2. Click **Refresh** to load profiles from the connected device
3. View all installed profiles in the table

### Provisioning a New Profile

1. Enter the activation code in the text field
2. Click **Provision**
3. Wait for the provisioning process to complete
4. The new profile appears in the list

### Activating a Profile

1. Select a profile from the list
2. Click **Activate**
3. The profile status changes to "Active"

### Deactivating a Profile

1. Select an active profile
2. Click **Deactivate**
3. The profile status changes to "Inactive"

### Removing a Profile

1. Select a profile from the list
2. Click **Remove**
3. Confirm the removal
4. The profile is deleted from the device

## Viewing System Logs

1. Navigate to **Logs** in the sidebar
2. View all system events and operations
3. Use **Clear Logs** to remove old entries

## Configuring Settings

1. Navigate to **Settings** in the sidebar
2. Configure connection preferences:
   - Auto-connect to last device
   - Enable offline recovery mode
3. Set enterprise configuration server URL
4. Click **Save Settings**

## Common Tasks

### Switching Between Devices

1. Go to Device Connection
2. Click **Disconnect** to disconnect current device
3. Discover and connect to a different device

### Troubleshooting Connection Issues

If you can't connect to a device:

1. Verify the device is powered on
2. Check WLAN/Bluetooth is enabled on your PC
3. Try discovering devices again
4. Check the Logs for error messages
5. See [Troubleshooting Guide](troubleshooting.md) for more help

### Monitoring Operations

- Watch the status bar at the bottom for real-time updates
- Check the Logs view for detailed operation history
- Dashboard shows current connection status

## Keyboard Shortcuts

- **F5**: Refresh current view
- **Ctrl+L**: Navigate to Logs
- **Ctrl+D**: Navigate to Dashboard
- **Ctrl+Q**: Quit application
